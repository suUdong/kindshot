## 4. 개별 전략 효과 분석

### 4.1 TP 조정 (1.5% -> 0.8%)

- Peak >= 1.5% 도달: 2/23건 (9%) — old TP가 실제 발동한 경우
- Peak >= 0.8% 도달: 3/23건 (13%) — new TP가 추가 1건 커버
- **87%의 거래가 0.8%에도 미도달** — TP 조정의 실질 영향은 제한적
- 2건의 TP 히트에서 각 -0.70% (1.50% -> 0.80%) 차이 발생

**평가**: TP 인하는 상방 이익을 제한하지만, 뉴스 반응 특성상 대부분의 거래가 0.8% 미만이므로 영향 범위 좁음. 단, 474610(RF시스템즈) 사례에서 peak +2.70% 후 close -9.50%로 급락 — **빠른 익절이 대형 손실 방어에 핵심**.

### 4.2 Trailing Stop

- 스냅샷 기반 시뮬레이션에서 1건만 발동 (017960 한국카본)
- 해당 건: t+5m -0.36%에서 trail 발동, 실제로는 t+30m에 +0.99%로 회복
- **스냅샷 간격이 너무 넓어 trailing stop 평가 불가능**

**평가**: 연속 가격 스트림에서만 정확히 평가 가능. 474610 사례(peak +2.70% -> close -9.50%)가 trailing stop의 존재 이유를 입증. 서버 런타임에서 실제 발동 데이터 수집 필요.
