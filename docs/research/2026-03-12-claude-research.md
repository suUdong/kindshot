# Claude Research: 뉴스 기반 자동 단타 시스템 생존 가이드

> 2026-03-12 Claude Research로 생성

## 1. "3건 전부 수익"이라는 착각이 위험한 이유

### 소표본 편향과 과신의 심리학

Tversky와 Kahneman의 고전 연구가 정확히 이 상황을 설명한다. 인간은 작은 표본에서 나온 결과를 전체 모집단의 특성으로 착각하는 강력한 인지 편향(소수의 법칙)을 가진다. 3건 전승은 50% 승률 전략에서도 12.5% 확률로 발생하는 평범한 사건이다.

통계적 유의성 확보 기준:
- 중심극한정리 기준 최소 시작점: 30건
- 기본적 통계 신뢰성: 100건 이상
- López de Prado(2018) 권장: 200~500건
- 95% 신뢰수준: 385건, 99% 신뢰수준: 666건

### 뉴스 트레이딩의 구조적 실패 패턴

UC San Diego Timmermann(2025) 연구에 따르면, 미국 시장에서 실적 발표 후 가격 반응은 밀리초 단위로 시작되며 5초만 늦어도 평균 수익률이 0.72%에서 0.41%로 급감한다.

현재 시스템은 3초 간격 폴링 + 키워드 분류 + LLM 판단 + API 주문이라는 파이프라인을 거치므로 뉴스 발표 후 실제 주문까지 최소 5~15초 이상 소요된다. 한국 시장은 미국보다 HFT 비중이 낮아 기회의 창이 존재하지만, 이 창은 해마다 좁아지고 있다.

### 백테스트와 실전의 괴리

Campbell Harvey와 Yan Liu(2014, Duke University)에 따르면, 20개 전략을 테스트하면 순전히 운으로 수익성 있는 전략을 찾을 확률이 64%에 달한다. 50개를 테스트하면 거의 확실하다.

과적합의 경고 신호: 최적화 파라미터 10개 초과, Sharpe ratio 3.5 이상, 하나의 시장·기간에서만 작동.

## 3. 한국 공시 유형별 진짜 돈이 되는 것과 함정

| 공시 유형 | 신호 방향 | 핵심 조건 |
|----------|----------|----------|
| 자사주 소각 | ★★★ 강한 매수 | 이익소각인지 확인, 감자와 구분 |
| 대규모 공급계약 | ★★ 매수 | 매출 대비 15% 이상 |
| 자사주 매입 | ★ 약한 매수 | 소각 약속 없으면 할인 |
| 무상증자 | ★ 약한 매수 | 극적 반응은 드뭄 |
| 특허 취득 | ★ 조건부 | KOSDAQ 바이오·테크에서만 유의미 |
| 유상증자 | ★★ 매도 | 규모 클수록 부정적 |
| CB 발행 | ★★ 매도 | 리픽싱 조항 있으면 희석 위험 극대화 |
| 최대주주 변경 | 방향 불확실 | 맥락 분석 필수 |

장중 공시는 당일 내에 가격에 완전히 반영된다(Lee, Choi 2009). 보유 시간이 길어질수록 알파가 소멸.

## 4. 리스크 관리

### 자금 관리 3대 철칙

1. **1% 규칙** (거래당): 총 자본의 1%를 최대 손실로 설정
2. **6% 규칙** (월간): 실현 손실 + 열린 포지션 리스크 합계가 6%면 신규 매수 중단
3. **일일 손실 한도**: -1% 도달 시 자동 거래 중단

### ATR 기반 포지션 사이징

```
포지션 수량 = (계좌 자본 × 1%) ÷ (ATR(5) × 2.0)
```

예시: 계좌 ₩10,000,000, 종목 가격 ₩50,000, ATR ₩1,200
- 거래당 최대 리스크 = ₩100,000
- 손절 거리 = ₩2,400
- 포지션 수량 = 41주

### 드로다운의 수학

- 10% 손실 → 11.1% 수익으로 복구
- 30% 손실 → 42.9% 필요
- 50% 손실 → 100% 필요

### 시장 폭락 안전장치

- 서킷브레이커: KOSPI -8%→20분 중단, -15%→추가 중단, -20%→당일 종료
- 사이드카: KOSPI200 선물 -5% 1분 지속 시 프로그램 매도 5분 정지
- VI: 개별 종목 ±3~6% 변동 시 2분간 단일가 매매

## 5. 시스템 고도화 로드맵

### 데이터 축적 단계별 분석

**1주차 (15~25건):** 기초통계, 버킷별 승률, 시간대별 분포, 슬리피지 추정, 버그 점검

**1개월 (60~100건):** Sharpe, Sortino, Profit Factor, Expectancy, 카테고리별 성과, 보유 시간 최적화, Kelly 분수 계산

**3개월 (200~400건):** Walk-forward 분석, Monte Carlo 시뮬레이션, 시장 레짐 분석, 전략 감쇠 모니터링

### 실전 전환 4단계

1. Phase 1 – 페이퍼 (4~8주): 100건+ 거래
2. Phase 2 – 마이크로 라이브 (1~2주): 최소 수량
3. Phase 3 – 소규모 라이브 (2~4주): 목표의 10%
4. Phase 4 – 점진적 스케일업: 2주마다 25%씩

페이퍼 성과의 50~70%를 실전 기대 수익으로 잡는 것이 현실적.

### 결론

수익 데이트레이더는 1~3% 미만(Barber et al. 2014, Chague et al. 2020). 돈을 버는 것보다 잃지 않는 것이 먼저이며, 시장에서 살아남는 것 자체가 첫 번째 알파다.
